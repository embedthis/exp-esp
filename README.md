exp-esp
===

Expansive plugin for ESP files.

Provides the 'compile-esp' service.

### To install:

    pak install exp-esp

### To configure in expansive.json:

* compile-esp.enable &mdash; Set to true to enable the compilation of ESP files.

```
{
    services: {
        'compile-esp': {
            enable: true
        }
    }
}
```
